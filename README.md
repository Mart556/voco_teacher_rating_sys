## Installation
1. ```npm install -g pnpm``` <br />
Tõmbab pnpm package, mida tuleb kasutada projekti package'ite installimiseks.

2. ```pnpm i``` <br />
Seda peab tegema, et saaks ilma liigsete failideta installida projekti package'id. Seda sammu peab nii 'frontend/' kui ka 'backend/' kaustades eraldi tegema.

3. "backend/" kaustas README juhendab mida peab lisaks veel tegema, et seda üles sättida.
